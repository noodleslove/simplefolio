EthStates

Ethstates is an innovative blockchain-based real estate transaction prototype that harnesses the power of blockchain's authenticity to simplify property transactions by removing intermediaries. Currently hosted on the Polygon testnet, it enables direct exchanges between buyers and sellers, drastically lowering costs and enhancing transparency, thereby making real estate investments more accessible and efficient for everyone involved.

NutriPal

NutriPal is an intuitive diet-tracking platform built with React and MongoDB, designed to streamline daily meal logging. It offers personalized nutrition insights and recommendations via ChatGPT API integration, enhancing user engagement and providing tailored dietary advice for a healthier lifestyle.

Pandemics and the Planet: Analyzing the Correlations Between COVID-19 and Global Warming

This project examines the connections between COVID-19 and global warming, exploring how pandemic-driven changes in human activity impacted environmental metrics like carbon emissions and air quality. The project analyzes data trends to highlight the intertwined effects of health crises and climate change on global sustainability.

The UCSD Computer Science and Engineering Society (CSES) website is a dynamic platform built with React, Node.js, and MongoDB, featuring Google authentication for secure access.
